# [WIP] hoost

> Easily manage your virtual hosts

[![NPM version][npm-version-image]][npm-version-url]
[![Build Status][travis-image]][travis-url]

## How to install

```sh
npm i -g hoost
```

## How to use

### `add [ip] [host]`

Add host on `/etc/hosts`.

```sh
[sudo] hoost add 127.0.0.1 yourhost.com
```
